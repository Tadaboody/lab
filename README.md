# Lab
A partial port of [hub](https://www.github.com/github/hub) but for gitlab

## Install
```sh
# Activate your favorite virtual envoriment
python3 setup.py install
```

## Features:
- [ ] List issues
- [ ] Create issues
- [ ] List pull requests
- [ ] Create pull requests
- [ ] Checkout pull requests
- [ ] Create repository
- [ ] Fork repository
